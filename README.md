# Acunetix12-Scan-xray
调用AWVS的接口实现任务添加、删除。使用AWVS的爬虫转发浏览至XRAY中。

### **功能：**
### **批量提交任务**
### **批量导出任务结果**
### **批量删除任务**

### **使用：**
### **按config.py中注释说明进行配置即可使用**

### **二次修改：**
 **修改了awvs使用crawlonly模式时的方式，添加了代理IP和端口，可以在config.py中xray_address和xray_port填写相关IP和端口**
 **api_key是awvs的api，安装完毕后自行查看**
 **host是awvs的地址**
